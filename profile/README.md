## 📌 About the App
**Betaflight Configurator** is the official desktop application for configuring Betaflight-powered flight controllers. It allows FPV drone pilots to fine-tune every aspect of their setup — from PID tuning and receiver mapping to OSD, motors, and sensors.

## ❓ What is Betaflight Configurator?
This utility connects your drone’s flight controller to your PC via USB, letting you modify flight parameters, update firmware, calibrate sensors, and visualize live data. Designed for hobbyists and professionals, it ensures precise control and stability for your quadcopter.

## ✅ Why Choose Betaflight Configurator?
- Official open-source utility from the Betaflight project  
- Simple interface for PID, rates, and filters tuning  
- Integrated firmware flasher for flight controllers  
- Real-time data monitoring and sensor graphs  
- OSD layout editor and receiver mapping tools  
- Full compatibility with popular FC hardware  
- Designed for FPV pilots and drone enthusiasts  

## 🔧 Key Features
- Flight controller setup and configuration via USB  
- PID, filter, and RC rate tuning modules  
- Firmware flashing (Betaflight targets)  
- Live motor testing and ESC calibration  
- On-screen display (OSD) customization  
- Sensor calibration and accelerometer setup  
- Blackbox log viewer and data export  
- CLI access for advanced commands  

---

## 🖥️ System Requirements
- Windows 10/11 (64-bit)  
- USB cable for FC connection  
- At least 200 MB free disk space  
- Internet connection for firmware downloads  

## ▶️ Usage
1. Download and install Betaflight Configurator.  
2. Connect your flight controller via USB.  
3. Select the COM port and click **Connect**.  
4. Configure tuning, receiver, OSD, and flight modes.  
5. Save settings and safely disconnect before flight.
